RFID Developer Kit for .NET

.NET Samples

Sample applications are designed to demonstrate the usage of assemblies available in RFID SDK for .NET. They are provided as a learning tool and for quick reference during application development.

Feature Sample Description Device Families Supported
RFID3 CS_RFID3_Host_Sample1

 		 Shows how to use Symbol.RFID3 assembly to perform basic inventory and single tag access operations (read/write/lock/kill). This sample is designed to run in a host, such as a PC, to remotely control RFID-enabled devices. Host (PC) based application : FX7400, FX7500, FX9500, XR series (LLRP enabled)
CS_RFID3_Host_Sample2
 Shows how to use Symbol.RFID3 assembly to perform advanced inventory and tag access operations (read/write/lock/kill) on single and multiple tags based on filters (pre-filters, post-filters and access-filters), reader capabilities and configuration and reader management functions. This sample is designed to run in a host, such as a PC, to remotely control RFID-enabled devices. Host (PC) based application : FX7400, FX7500, FX9500, XR series (LLRP enabled)
