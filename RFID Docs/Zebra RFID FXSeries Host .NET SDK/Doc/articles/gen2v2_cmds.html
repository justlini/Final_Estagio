Gen2 V2 Commands

Readers which support Gen2 V2 commands report the same in the field IsG2V2CommandSupported of ReaderCapabilities as true. The following operations can be performed on Gen2 V2 commands in TagAccess class:

  1. AuthenticateWait / AuthenticateEvent - This API allows reader to perform mutual authentication with tag

    TagData readAccessTag;

TagAccess.AuthenticateAccessParams authenticateParams = new TagAccess.AuthenticateAccessParams();

authenticateParams.SenResp = true;

authenticateParams.IncRespLen = true;

authenticateParams.CSI = 0;

authenticateParams.MessageBitLength = 96;

byte[] TAM1 = new byte[12] { 0x00, 0x00, 0x96, 0x56, 0x44, 0x02, 0x37, 0x57, 0x96, 0xC6, 0x96, 0x64 };

authenticateParams.Message = TAM1;

authenticateParams.AccessPassword = 1;

string gen2TagID = "DADACAFEDADACAFEDADACAFE";

readAccessTag = rfid3.Actions.TagAccess.AuthenticateWait(gen2TagID, authenticateParams, null);

  2. ReadBufferWait / ReadBufferEvent - This API allows reader to read data stored in a Tag’s ResponseBuffer.

    TagData readAccessTag;

TagAccess.ReadBufferAccessParams readBufferAccessParams = new TagAccess.ReadBufferAccessParams();

readBufferAccessParams.WordPtr = 0;

readBufferAccessParams.BitCount = 0x80;

readBufferAccessParams.AccessPassword = 1;

readAccessTag = rfid3.Actions.TagAccess.ReadBufferWait(gen2TagID, readBufferAccessParams, null);

  3. UntraceableWait / UntraceableEvent - This API performs Untraceable command and allows to reader alter U bits in EPC memory, hide memory (EPC, TID, user memory), reduce operating range.

    TagData readAccessTag;

TagAccess.UntraceableAccessParams untraceParams = new TagAccess.UntraceableAccessParams();

untraceParams.AssertU = true;

untraceParams.HideEPC = true;

untraceParams.HideTID = TID_HIDE_STATE.TID_HIDE_SOME;

untraceParams.HideUser = false;

untraceParams.OperatingRange = TAG_OPERATING_RANGE.RANGE_REDUCED;

untraceParams.EPCLength = 3;

readAccessTag = rfid3.Actions.TagAccess.UntraceableWait(tagsInField[nIndex], untraceParams, null);

  4. CryptoWait / CryptoEvent - This API performs crypto function with tag

    TagData readAccessTag;

TagAccess.CryptoAccessParams cryptoParams = new TagAccess.CryptoAccessParams();

cryptoParams.IChallenge[0] = 0x96564402;

cryptoParams.IChallenge[1] = 0x2375796C;

cryptoParams.IChallenge[2] = 0x96640000;

cryptoParams.KeyID = 0;

cryptoParams.CustomData = false;

cryptoParams.Profile = 0;

cryptoParams.Offset = 0;

cryptoParams.BlockCount = 0;

cryptoParams.ProtMode = 0;
